	[ 참고 : 언리얼 엔진 4 블루프린트 게임 만들기. ]

-----------------------------------
1]	개요.
-----------------------------------

	-	로켓이 목표물과 충돌할 때 연출되는 폭파 이펙트는 크기를 더 키워서
		시각적인 효과가 충분히 나타나도록 함.

	-	[ P_Exp_2 ]에  적용된 [ 이미터 ]를 모두 수정.


-----------------------------------
1]	모듈의 기능.
-----------------------------------

	[ 참고 : https://docs.unrealengine.com/4.27/ko/RenderingAndGraphics/ParticleSystems/ ]

		주요 모듈				|		개요.
	-------------------------------------------------------------
		Spawn					|	입자의 수와 방출방식 설정.
		Lifetime				|	입자의 수명, 단위 : 초
		Color Over Life			|	시간의 흐름에 따른 입자의 색상 설정.
		Size By Life			|	시간의 흐름에 따른 입자의 크기.
		Dynamic					|	다이나믹 파라미터 생성.
		Initial Rotation Rate	|	초기 회전 속도.
		Initial Rotation		|	초기 회전 각도.
		SubImage Index			|	여러개의 이미지로 구성된 경우 이미지의 인덱스 번호.
		Const Acceleration		|	가속도 방향.
		Sphere					|	입자의 방출이 시작될 가상의 구의 영역 크기.
		Collision				|	입자의 충돌시 반사 여부.
		Light					|	파티클에 적용할 조명.
		Initial Velocity		|	초기 속도.
		

		

-----------------------------------
2]	[ ShockWave ] 키우기
-----------------------------------

	-	
		모듈명			|	속성																|	값			|	비고					
		-----------------------------------------------------------------------------------------------------------------------------
		Lifetime		|	[ 수명 ] > [ Lifetime ] > [ Distribution ] >						|				|
						|	[ Distribution Float Constant ]									|	1.0			|	입자의 수명( 단위 : 초 )
		-----------------------------------------------------------------------------------------------------------------------------
		Size By Life	|	[ 크기 ] > [ Life Multiplier ] > [ Distribution ] >				|				|
						|	[ Distribution Vector Constant Curve ] > [ Constant Curve ] >	|				|	시간에 따른 입자 크기.
						|	[ Points ] > [ 0 ] > [ Out Val ]								|	[ 5, 5, 5 ]	|
						|				 [ 1 ] > [ Out Val ]								|	[ 8, 8, 8 ]	|
		-----------------------------------------------------------------------------------------------------------------------------

3]	[ FireBall ] 키우기
-----------------------------------

	-	
		모듈명			|	속성																|	값				|	비고					
		----------------------------------------------------------------------------------------------------------------------------
		Lifetime		|	[ 수명 ] > [ Lifetime ] > [ Distribution ] >						|					|
						|	[ Distribution Float Uniform ] > [ Min ] / [ Max ]				|	1 / 1.5			|
		----------------------------------------------------------------------------------------------------------------------------
		Size By Life	|	[ 크기 ] > [ Life Multiplier ] > [ Distribution ] >				|					|
						|	[ Distribution Vector Constant Curve ] > [ Constant Curve ] >	|					|
						|	[ Points ] > [ 0 ] > [ Out Val ]								|	( 20, 20, 20 )	|
		----------------------------------------------------------------------------------------------------------------------------

4]	[ Sparks ] 키우기
-----------------------------------

	-	
		모듈명			|	속성																|	값				|	비고					
		----------------------------------------------------------------------------------------------------------------------------
		Lifetime		|	[ 수명 ] > [ Lifetime ] > [ Distribution ] >						|					|
						|	[ Distribution Float Uniform ] > [ Min ] / [ Max ]				|	0.8 / 1			|
		----------------------------------------------------------------------------------------------------------------------------
		Initial Size	|	[ 크기 ] > [ Start Size ] > [ Distribution ] >					|					|
						|	[ Distribution Vector Constant ] > [ Constant ] 				|	( 20, 200, 0 )	|	입자의 초기 크기.
		----------------------------------------------------------------------------------------------------------------------------
		Sphere			|	[ 위치 ] > [ Velocity ]											|	Off				|	구체의 표면에서만 발산.
						-----------------------------------------------------------------------------------------------------------
						|	[ 위치 ] > [ Start Radius ] > [ Distribution ] >					|	( 30, 50 )		|	발산 속도.
						|	[ Distribution Float Uniform ] > [ Min ] / [ Max ]				|					|
		----------------------------------------------------------------------------------------------------------------------------

-----------------------------------

	-
		모듈명			|	속성																|	값				|	비고

						|	[ Distribution Float Uniform ] > [ Min ] / [ Max ]				|		1 / 2		|
