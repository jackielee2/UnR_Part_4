11. 마우스로 화면 각도 조절하기.
	[ 참고 : 언리얼 엔진 4 블루프린트 게임 만들기. ]
	-	위 책의 내용을 바탕으로 게임을 만들면서 언리얼을 익힙니다.
==================================================================
-----------------------------------
0]	개요
-----------------------------------

	-	마우스를 드래그하여 카메라로 헬기의 위나 아래에서
		게임 화면을 보는 기능 추가.

	-	마우스 드래그량을 스프링암의 회전 각도로 변환.



-----------------------------------
1]	마우스 입력 장치 추가.
-----------------------------------

	-	Axis 매핑 추가.

			매핑 이름		|	장치			|	비고
		--------------------------------------------------------------------
		Mouse Rot LR		|	마우스 X		|	화면의 좌우 각도 조정.
		--------------------------------------------------------------------
		Mouse Rot UD		|	마우스 Y		|	화면의 상하 각도 조정.
		--------------------------------------------------------------------



-----------------------------------
2]	화면 스크롤 함수 만들기.
-----------------------------------

	-	방식.

		1)	마우스 상하 드래그.

		2)	1) 값을 [ ArmPitch ] 변수에 누적.
		
		3)	2) 값을 스프링 암의 Pitch로 설정.



	-	함수 생성.

		-	이름			:	[ ScrollUpDown ]
			입력 - 이름	:	[ Axis Value ]
				 - 타입	:	[ Float ]


	-	이하 [ ScrollUpDown ] 참고.
