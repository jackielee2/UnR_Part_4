6-2. 입력 장치 설정
	[ 참고 : 언리얼 엔진 4 블루프린트 게임 만들기. ]
	-	위 책의 내용을 바탕으로 게임을 만들면서 언리얼을 익힙니다.
==================================================================
-----------------------------------
0]	개요
-----------------------------------

	-	헬기를 이동시키기 위한 입력 장치 설정.
	-	[ 프로젝트 세팅 ] > [ 입력 ]에서 설정.
	-	방식
		-	[ 액션 매핑 ]
			-	누를 때마다 한 번씩 동작하는 입력 설정.
			-	점프, 총알 발사 등.

			[ 축 매핑 ]
			-	누르고 있으면 계속해서 동작하는 입력 설정.
			-	이동, 회전 등.


-----------------------------------
1]	매핑 설정.
-----------------------------------

	-	개요.
	
		-	[ + ] 버튼 클릭 후
			매핑할 이름, 매핑에 적용할 입력 장치 설정.

			-	매핑 이름
				-	공백 가능.
				-	한글 사용 가능.

		-	설정한 매핑 정보는 [ 블루프린트 액션 목록 ]의 [ 입력 카테고리 ]에 표시됨.

		-	매핑 정보는 [ Config 폴더 ] > [ DefaultInput.ini ] 파일에 저장됨.
			-	다른 프로젝트의 [ Config 폴더 ]에 [ DefaultInput.ini ] 파일을 복사하면
				동일한 키 매핑 사용 가능.



	-	설정.

		1)	액션 매핑 설정.

			매핑 이름		|	장치					|	비고
			-----------------------------------------------------------------
			Shoot			|	왼쪽 Ctlr			|	로켓 발사 키
							|	왼쪽 마우스 버튼		|	로켓 발사 버튼.
			-----------------------------------------------------------------
			Convert View 	|	X					|	1,3인칭 시전 전환.
			-----------------------------------------------------------------
			Popup Menu		|	M					|	게임의 팝업 매뉴 호출
			-----------------------------------------------------------------



		2)	축 매핑 설정.

			매핑 이름			|	장치			|	Scale		|
			------------------------------------------------------
			Move Forward Back	|	W / S		|	1 / -1		|	
			------------------------------------------------------
			Move Left Right		|	A / D		|	-1 / 1		|
			------------------------------------------------------
			Move Up Down		|	Z / C		|	-1 / 1		|
			------------------------------------------------------
			Rotate				|	Q / E		|	-1 / 1		|
			------------------------------------------------------
