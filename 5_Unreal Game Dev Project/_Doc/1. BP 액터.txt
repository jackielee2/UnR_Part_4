	--------------------------
	-	C++ 클래스와 BP 클래스
		사이의 차이..
	--------------------------

		-	C++ 클래스를 이용한 방식이
			BP 클래스를 생성하는 방식보다
			더 고급 기능 사용 가능..


		-	BP 클래스가 C++ 클래스에 비해
			작업이 더 쉬움..

			ㄴ	BP 클래스는 3D 메시나 트리거와 같은
				클래스들의 시각적 컴포넌트를
				쉽게 살펴보고 편집 가능하며
				에디터에 노출된 C++ 클래스에 정의된
				속성도 변경 가능..


		-	C++ 클래스로 작성한 로직이
			BP로 작성한 로직에 비해
			속도가 더 빠름..


		-	소스 버전 관리 플랫폼에서
			두명 이상의 작업자가 하나의 
			소스에 접근하여 작업할 때
			BP가 충돌 가능성이 높음..
			
			ㄴ	BP 클래스는 텍스트 파일 대신
				바이너리 파일로 변환됨..


		-	BP 클래스는 C++ 클래스나
			다른 BP 클래스를 상속 가능..

			ㄴ	C++ 클래스에 정의된 함수를
				호출 하거나 재정의 가능..
